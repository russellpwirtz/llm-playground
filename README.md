# llm-playground
Testbed for training / fine-tuning / querying LLMs

Steps:
### Setup
- Find a model on huggingface
- Find data to include
### Fine Tune
- Run fine-tune.py
- Run copy-files.py
- Run query-fine-tune.py
### Tune the tuned
- Run train-tuned.py
- Run copy-files.py
- Run query-fine-tune.py
